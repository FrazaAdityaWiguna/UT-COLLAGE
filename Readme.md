## Cara menjalankan (umum)

1. Pastikan JDK terpasang (versi 8+ direkomendasikan).
2. Buka terminal di direktori yang berisi file `.java` yang ingin dijalankan.
3. Kompilasi dan jalankan:

```
javac answer1.java
java answer1
```

Catatan: Jika kelas berada dalam package, jalankan dari root project dan gunakan opsi path/package yang sesuai.
